Qué ventajas ofrece CSS

Los CSS aportan un mejor control en la presentación de un sitio web que los elementos de HTML, por ejemplo.
Más accesibles: Los CSS permiten modificar el formato de una página web según sean las necesidades del usuario. Por ejemplo, personas con deficiencias visuales.

¿Qué es CSS y para qué sirve?

CSS es un lenguaje utilizado en la presentación de documentos HTML. Un documento HTML viene siendo coloquialmente “una página web”
Entonces podemos decir que el lenguaje CSS sirve para organizar la presentación y aspecto de una página web.
